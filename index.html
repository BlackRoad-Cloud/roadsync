roadsync.config.ts 
// RoadSync — conflict-free file synchronization
import { RoadSync } from '@blackroad/roadsync';

const sync = new RoadSync({
  storage: {
    backend: 'minio',             // self-hosted object storage
    endpoint: 'cecilia.local:9000',
    buckets: ['docs', 'media', 'backups', 'configs'],
  },
  encryption: {
    algorithm: 'xchacha20-poly1305',
    keyDerivation: 'argon2id',
    e2e: true,                   // server never sees plaintext
  },
  conflict: {
    strategy: 'crdt',             // conflict-free replicated data types
    vectorClock: true,            // causal ordering
  },
  offline: true,                   // works without internet
});

// Sync starts automatically, merges happen in background
await sync.watch('~/Documents');
// Changes propagate to all devices within 100ms on LAN

Sync that respects your data

Your files belong to you. Not a cloud provider. Not an ad network.

🔄

CRDT Conflict Resolution

Conflict-free replicated data types ensure that concurrent edits on different devices merge automatically without data loss. No "conflicted copy" files cluttering your folders ever again.

🔒

End-to-End Encryption

XChaCha20-Poly1305 encryption with Argon2id key derivation. Your sync server never sees plaintext. Even if someone compromises the storage backend, your files remain encrypted and unreadable.

⚡

Local-First Architecture

Every device maintains a complete local copy. Edit files offline for days, connect when you're ready, and changes merge seamlessly. No internet required for read or write operations.

📦

MinIO Object Storage

Self-hosted S3-compatible object storage on your own hardware. 4 buckets organizing docs, media, backups, and configs. Scale storage by adding drives, not by paying monthly fees.

🔍

Version History

Every file change is versioned with vector clocks for causal ordering. Restore any previous version instantly. Understand exactly when and where each change originated across your device fleet.

🌐

Cross-Platform

Native clients for macOS, Linux, Windows, iOS, and Android. Raspberry Pi sync nodes for always-on edge replication. CLI for automation and scripting. GUI for when you just want to drag and drop.
